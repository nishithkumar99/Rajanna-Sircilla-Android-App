- Integrated a suite of external APIs (e.g., Google Maps/Location Services, Push Notification Services) to provide core functionality, including:
      -Real-time directions and points-of-interest (POI) discovery.
      -Dynamic content updates (e.g., event listings, administrative announcements).
      -Location-based alerts for nearby services.
- Applied Material Design principles to create a highly intuitive and compelling user interface (UI), focusing on seamless navigation and an optimal user experience (UX) across various Android devices.
- Engineered comprehensive service integration for quick access to essential local information, such as nearest police stations, hospitals, tourism sites, and administrative offices, thereby improving accessibility for users unfamiliar with the district
